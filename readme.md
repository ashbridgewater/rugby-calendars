# 🏉 Rugby Calendars — Autumn Internationals 2025

Automatically generated `.ics` calendar files for the **2025 Autumn Internationals**.  
All fixtures are compiled from [autumn-internationals.co.uk](https://www.autumn-internationals.co.uk/2025/) and rebuilt through a GitHub Actions workflow whenever the fixture list changes.

---

## 📅 Subscribe to the Calendar

You can subscribe to the calendar directly in any app that supports `.ics` URLs:

**Raw subscription link:** https://raw.githubusercontent.com/ashbridgewater/rugby-calendars/main/calendar/autumn_internationals_2025.ics

### Apple Calendar (macOS / iOS)
1. Go to **File → New Calendar Subscription…**  
2. Paste the link above.  
3. Choose **Location: iCloud** (to sync across devices).  
4. Set **Auto-refresh** to *Hourly* or *Daily*.  
5. Click **OK**.

### Google Calendar (Web)
1. In the sidebar, under **Other calendars**, click the **+** icon.  
2. Choose **From URL**.  
3. Paste the link above.  
4. Click **Add calendar**.

### Outlook (Web / Desktop)
1. **Add calendar → Subscribe from web**.  
2. Paste the link above.  
3. Name it (e.g. “Autumn Internationals 2025”).  
4. Click **Subscribe**.

Your app will refresh automatically whenever the `.ics` file updates.
